Intuitive GUI

The GUI provides an intuitive and simple interface for developing Java programs. This greatly simplifies and eases the programmmer’s job.

Graphical Class Representation

The application classes are represented in the form of boxes. Each box contains information such as the class name, the variables, and methods contained therein. Double clicking on the box displays the Class Properties Dialog Box. A right mouse click on a box displays an menu of items that may be associated. The menu contains the following options:-

  • Add Variable - Displays a dialog box where the user can enter a new variable name and its properties.
  • Edit Variable Properties - Displays a dialog box where the user can edit the properties of the various variables associated with the object.
  • Add Method - Displays a dialog box where the user can input a new method and the properties associated with the method.
  • Edit Method Properties - Displays a dialog box where the user can edit the properties associated with the various methods contained in the object.
  • Edit Method Body - Displays a dialog box where the user can edit the body of a method associated with this class.

The hierarchy of the various objects is displayed by means of directed links, making the task of understanding an application’s design much simpler.

Drag and Drop Features

The tool provides Drag and Drop Features, i.e., a user can create an object and create it’s links by means of a mouse.

Code Generation

Brewery outputs Java code files based on the graphical representation developed above. It will output compilable code.

Document Generation

Brewery also outputs comments for methods and classes in the Javadoc format.
