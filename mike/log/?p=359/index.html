Rails is Back!

Well, I’ve hemmed and hawwed (?) the other way now: Rails is back. I’ve given up on Java and restarted my project in Ruby on Rails. Now I have a decent IDE in Radrails so I’m hopeful that I will be a little more productive than on the Mac Mini with vi. I can already see some amazing productivity improvements - bug fixing is painless with Ruby’s complete lack of a deployment step. It took me about 10 minutes to create and debug a login page with the code located at the bottom of this page.
