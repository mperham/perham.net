Thoughts on Fedora

I’ve been setting up Fedora on my home PC on and off for the last week or so. I am really impressed so far. Maybe all the distros have good package management now but the addition of YUM to the system makes software discovery and installation trivial. I wanted a Gnome applet that would notify me when I got a GMail so I did a Google search and found Mail Notification. Once I verified that it looked like it would solve my problem, I just ran “yum install mail-notification” and everything was downloaded and installed for me without a single asinine license to accept or install wizard to plow through mindlessly. It Just Works. Brilliant.

On the down side, Linux still sucks at Power Management. Seriously Linux guys, I’ve been hibernating and sleeping my desktops in Windows for the last 6-8 years now and you guys still can’t get it working? Instead I have the awful choice of shutting down my system every night or leaving it on all day. Would you please make it easy to be environmentally friendly?
