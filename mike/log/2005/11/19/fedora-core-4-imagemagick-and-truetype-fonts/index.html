Fedora Core 4, ImageMagick and TrueType fonts

November 19th, 2005 · 6 Comments

I just wasted the last 2 hours trying to get RMagick to work properly on my Fedora Core 4 desktop. Apparently the ImageMagick 6.2.2 package that ships with FC4 is simply broken and points to an Arial font that does not exist on my machine. If you get a bunch of font-related errors when trying to run ‘gem install rmagick’ you probably have the same problem. To fix it, I just had to download and install the latest ImageMagick base and devel packages. Once installed, just run ‘gem update rmagick’ and you should find the problem gone. If this solves your problem, leave me a comment and say so!
