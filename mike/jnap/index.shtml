Welcome to the Jnap Home Page

Jnap is a pure-Java clone of the Napster mp3-sharing system.

Update - 6/1/2001

Jnap will no longer be updated as Napster is moving to a subscription service. 1.1pr is the last official version that will be supported. Source code is still available via sourceforge as noted below. The latest source has several features - like preliminary support for OpenNap servers - that the 1.1pr version does not.

Download - Updated 10/28/2000

  • Jnap v1.1 Preview for Windows and Linux (150k)
Jnap requires Java 1.2.2 or better to run properly. Please make sure you have installed Java 1.2.2 on your system before attempting to run Jnap. You can download the latest version here.

Please note that this release is a preview release. It is stable but still has a few bugs left. Feedback, especially helpful feedback, is welcome.
