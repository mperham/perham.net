Mike's Unix Utility Collection for Windows

As a software developer, I find myself constantly using various Unix utilities as part of my daily work. These tools range from vim for basic text editing to ssh and scp for work with remote Linux servers. I have collected these tools into a single package which can be placed on any Windows machine.

What's Included

In no particular order:

  • grep, tail, head, sort, uniq and many other 'basic' utilities.
  • tar, gzip, zip, unzip
  • TightVNC viewer
  • gvim 6.2
  • ssh, scp, sftp, wget
  • cvs
  • nmap (you do know which ports are publicly visible on your servers, right?)
  • IfranView image viewer and editor
A complete listing can be found here.

Installation

  1. Unpack zip to c:\bin or your directory of choice.
  2. Put c:\bin in your PATH (i.e. set PATH=c:\bin;%PATH%)
